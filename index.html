Welcome to Scrum Project

Actually, this project is not complete enough. I created this project just to record the meeting result from scrum. Every morning, we set a scrum meeting and discuss about the project. And we define who have to work what. At this time, we can record these data. If you interest, you can get my project by the following command.

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout .
